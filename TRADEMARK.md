[OpenUsage](https://github.com/robinebers/openusage) (macOS, Swift), created by Robin Ebers. The
source code of both projects is MIT-licensed, so porting the design is permitted, but the two
projects are separate:

See `PORTING_NOTES.md` for a detailed, ongoing account of exactly what was ported, adapted for
Windows, or intentionally left out.

The "AIUsage" / "AIUsage.NET" name and any associated logo are used to identify this project.

## Forks of AIUsage.NET

If you fork AIUsage.NET, you must:

1. Choose a different name for your fork
2. Remove any AIUsage.NET-specific branding
3. Clearly state that your project is a fork and not the official AIUsage.NET
